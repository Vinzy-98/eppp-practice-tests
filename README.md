## How to Use

### For Untimed Practice:
1. Open `index.html` in your web browser
2. Select a test from the list
3. Answer questions at your own pace
4. Click "Check Answers" when complete
5. View your score and download results

### For EPPP Format (Timed):
1. Open `index.html` and click "Try EPPP Format Tests"
2. Or directly open `EPPP_Format/index_eppp.html`
3. Select a test (timer starts automatically)
4. Answer questions using navigator to jump around
5. Mark questions for review if needed
6. Review your answers before submitting
7. Download detailed results after completion
