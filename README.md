## How to Use

1. Open `index.html` in your web browser
2. Select a test from the list
3. Answer questions by clicking on your choice
4. Click "Check Answers" when complete
5. View your score and download results

## Test Features

- Select and modify answers before checking
- Visual feedback (green = correct, red = incorrect)
- Progress bar showing completion percentage
- Detailed results showing all questions and answers
- Download results as a text file for your records
